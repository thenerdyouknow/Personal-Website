Projects
Polling Application Using Secure Multiparty Computation:(In Progress)
Building a polling application which will allow users to create and host polls, allow people to vote on those polls, but the votes are obscured using Shamir Secret Sharing. Everyone involved in the process see the result of the poll but none of the intermediate information, not even the servers storing the votes. This is a way to securely compute a function(in this case addition), without leaking any confidential information.

Chess Position Predictor:

Upto Triple Error Correcting Codes in ATMega328:
Implemented upto Triple ECCs on a stock ATMega328 microcontroller using a variation of the Peterson-Gorenstein-Zierler algorithm(for single and double ECC) and Hamming(for single ECC) using Arduino based on C and C++. Optimized the implementation to run on less than half the Flash(32k bytes) and SRAM(2k bytes) memory available.
